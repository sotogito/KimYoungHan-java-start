# Collection
컬렉션의 주요 인터페이스로는 List, Set, Map 이있다. 
## List

- 순서를 유지한다.
- 중복 저장이 가능하다.
1. ArrayList
2. Vector
3. LinkedList

## Set

- 순서를 유지하지 않는다.
- 중복 저장이 불가능하다.

1. HashSet
2. TreeSet

## Map

- [key-value] 의 형태, 엔트리로 저장된다.
- key의 경우 중복 저장이 되지 않는다.
-> 새로운 값이 기존의 값을 덮어 씌움

1. HashMap
2. Hashtable
3. TreeMap
4. Propertiesㅈ
